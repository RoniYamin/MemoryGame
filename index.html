Color Memory Game

Game Rules

1) Choose your user name.

2) Pick the amount of pairs you want to play with (30 maxium).

3) Press "start" to start the game.

4) The goal of the game is to match all the pairs of cards.

5) Gameplay: Once you start the game, glip over two cards at a time. If they match, they stay exposed. If they don't match, flip them back face-down.

6) Memorization: Remember the positions of the cards you've flipped to make better matches in subsequent turns.

7) Winning: The game is won when all pairs have been matched.

8) Press "play again" to start a new game, and next to that button you can choose amout of pairs again.

That's it! Enjoy playing the memory game on your own and challenge your memory skills!
